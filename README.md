# 🌐 Web Programming 1 – Final Project (AI Explorer)

This repository contains the **Final Project** completed in the Web Programming 1 course.  
It is a responsive website named **AI Explorer** designed to introduce students to **Artificial Intelligence (AI)**, explain its history, showcase applications, and provide interactive learning through a quiz and contact form.  
The project integrates HTML, CSS, and JavaScript to provide a modern, educational user experience.

---

## 📁 Folder Structure

| Folder/File Name       | Description                                                                 |
|-------------------------|-----------------------------------------------------------------------------|
| HTML/Index.html         | Homepage with hero section, quick overview of AI applications               |
| HTML/About.html         | Page explaining what AI is and its history                                  |
| HTML/Applications.html  | Page showcasing AI applications in healthcare, education, transport, finance|
| HTML/Quiz.html          | Interactive quiz to test knowledge about AI                                |
| HTML/Contact.html       | Contact page with form for feedback and questions                          |
| CSS/Style.css           | CSS file for styling layout, colors, fonts, and responsiveness              |
| JS/Script.js            | JavaScript file for interactivity and quiz logic                            |
| Images/                 | Image assets (robot, ai-brain, healthcare, education, transport, finance)   |

---

## 🧠 Features & Topics Covered

- **Header & Navigation**: Consistent navigation across all pages  
- **Home Page**: Hero section with introduction and quick links  
- **About Page**: Definition of AI and timeline of its history  
- **Applications Page**: Cards showcasing AI in healthcare, education, transport, and finance  
- **Quiz Page**: Interactive quiz to test knowledge of AI concepts  
- **Contact Page**: Form for user feedback and communication  
- **CSS Styling**: Responsive design, layout grids, hover effects, and typography  
- **JavaScript Interactivity**: Quiz logic, form validation, and dynamic behaviors
